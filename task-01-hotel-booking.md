# Бизнес-функция: бронирование отеля

## 1. глоссарий
Пользователь: клиент сервиса, осуществляющий поиск и бронирование отеля.

Бронирование: процесс резервирования номера на определённый период времени.

Гость: лицо, для которого оформляется бронирование.

Сервис: информационная система, предоставляющая возможность поиска и бронирования отелей.

Фильтр: набор параметров для уточнения результатов поиска.

## 2. бизнес-цели
Целью бизнес-функции "бронирование отеля" является предоставлением пользователям возможности самостоятельного поиска и бронирования номеров в отелях через сервис туроператора.

Для достижения данной цели необходимо:

- увеличить количество бронирований отелей онлайн;
- сократить время бронирования отеля пользователем;
- повысить удовлетворённость пользователей удобством пользования сервиса;
- снизить нагрузку на саппорт за счёт автоматизации бронирования;
- обеспечить своевременное уведомление пользователя о подтвержении бронирования.
