# Бизнес-функция: бронирование отеля

## конкурентный анализ
Для анализа выбраны сервисы Яндекс Путешествия, Авито и Суточно.ру
|Сервис|Основные возможности|
|Яндекс Путешествия|Поиск отелей, фильтрация, отзывы, бронирование онлайн|
|Авито|Поиск жилья, фильтрация, отзывы, связь с владельцем|
|Суточно.ру|Поиск жилья, фильтрация, отзывы, связь с владельцем, бронирование онлайн|

## 1. глоссарий
Пользователь: клиент сервиса, осуществляющий поиск и бронирование отеля.

Бронирование: процесс резервирования номера на определённый период времени.

Гость: лицо, для которого оформляется бронирование.

Сервис: информационная система, предоставляющая возможность поиска и бронирования отелей.

Фильтр: набор параметров для уточнения результатов поиска.

## 2. бизнес-цели
Целью бизнес-функции "бронирование отеля" является предоставлением пользователям возможности самостоятельного поиска и бронирования номеров в отелях через сервис туроператора.

Для достижения данной цели необходимо:

- увеличить количество бронирований отелей онлайн;
- сократить время бронирования отеля пользователем;
- повысить удовлетворённость пользователей удобством пользования сервиса;
- снизить нагрузку на саппорт за счёт автоматизации бронирования;
- обеспечить своевременное уведомление пользователя о подтвержении бронирования.

## 3. описание бизнес-процесса
1. Пользователь переходит на главную страницу сервиса бронирования отелей.
2. Пользователь вводит город назначения, даты проживания и количество гостей.
3. Система выполняет поиск и отображает список доступных отелей.
4. Пользователь применяет фильтры для уточнения результатов поиска.
5. Пользователь выбирает подходящий отель и номер.
6. Система отображает подробную информацию о выбранном варианте размещения.
7. Пользователь подтверждает данные гостей и бронирует номер.
8. Пользователь оплачивает бронирование одним из доступных способов.
9. Система создаёт бронирование и отправляет пользователю уведомление об успешном бронировании.

## 4. user story
- как пользователь сервиса, я хочу найти отель по городу, датам проживания и количеству гостей, чтобы подобрать подходящий вариант размещения для своей поездки;
- как пользователь сервиса, я хочу использовать фильтры, чтобы быстрее найти отлеь, соответствующий моим требованиям по цене, рейтингу и условиям проживания;
- как пользователь сервиса, я хочу забронировать выбранный номер в отеле, чтобы гарантировать наличие места для проживания на время поездки;
- как пользователь сервиса, я хочу оплатить бронирование онлайн, чтобы завершить оформление заказа без обращения к саппорту;
- как пользователь сервиса, я хочу получить подтверждение бронирования после оплаты, чтобы не сомневаться в успешности оформления заказа.

## 5. ограничения
- доступно бронирование только тех отелей, которые размещены на сервисе туроператора;
- оплата доступна только поддерживаемыми сервисом способами;
- сервис должен учитывать наличие свободных мест на выбранные даты;
- поиск отелей доступен только по указанным туроператором направлениям;
- отмена бронирования возможна исключительно не менее чем за 7 календарных дней до даты заселения.
